NomadMap.io

Local Coffee, Wifi, and Co-Working Information for Digital Nomads

About

NomadMap is a community-driven application looking to answer the question, “Where can I work today?” This iteration is currently an MVP proof-of-concept and should be considered a work-in-progress.

Getting Involved

See the guide if you'd like to contribute your own favorite places to work.

If you're interested in getting involved in development work, send an email to rob@atomidesign.com or put in a PR on Github
